Tokenomics

To maintain the long-term value of the token supply, we have a deflationary plan to take out tiny bits of the 1 Billion token supply through buybacks and burn.

Taxes Destribution

2% Burn

2% of each transaction is allocated to the Burn Wallet to create a true burn, meaning you can exchange the actual value for the tokens you sent to the burn wallet

3% Rewards

3% of each transaction is redistributed to our $ ANTQ holders in XRP (bep20).

2% Buyback

2% of each transaction will be automatically made a Buyback to make sure the token has stable growth in order to create a rising price floor and stability.

3% Marketing

3% of each transaction will be sent to the marketing wallet which will help the team with sustaining organic growth and hype for the token.

1% Charity

1% of each transaction will be allocated as charity fee

1.5% Blog distribution

1.5% of each transaction is distributed to blog Authors

1.5% Development of the marketplace

1.5% of each transaction will be used for the marketplace development
